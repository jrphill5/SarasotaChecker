# SarasotaChecker

## Required Python3 Modules

* scrapy
* requests
* deepdiff

## Configuration Notes

* **auth.py** must be created and completed as follows:

    ```
    EMAIL_HOST = ''
    EMAIL_PORT = 0
    EMAIL_HOST_USER = ''
    EMAIL_HOST_PASSWORD = ''
    EMAIL_FROM = 'FromName <fromuser@domain.tld>'
    EMAIL_TO = ['ToName1 <touser1@domain.tld', 'ToName2 <touser2@domain.tld>']
    ```
